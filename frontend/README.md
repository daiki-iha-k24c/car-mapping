# ナンバー地域コンプリート

日本全国の自動車ナンバープレート地域を集めてコンプリートするWebアプリです。  
訪れた地域を記録して、地図を塗りつぶしていくことができます。

---

## 使い方

1. 画面上部の検索バーから地域名を検索します  
   - 例：`し` → `品川`
2. 候補を選択すると、記録用のモーダルが表示されます
3. 記録すると、地図や都道府県一覧に反映されます
4. 地図上の都道府県をタップすると、その県に存在するナンバー地域一覧を確認できます  
   （※ この画面では登録はできません）

---

## データの保存について（重要）

このアプリの記録データは、**使用している端末・ブラウザごと**に保存されます。

- 記録はそれぞれのスマートフォン・ブラウザ内に保存されます
- 同じURLを他の人が使っても、記録が共有されることはありません
- ブラウザの履歴やサイトデータを削除すると、記録は消えます
- 別の端末や別のブラウザには記録は引き継がれません

※ 本アプリはログイン機能を持たないため、クラウド同期は行われません。

---

## 推奨環境

- スマートフォン（iOS / Android）
- モバイルブラウザ（Safari / Chrome）

※ ホーム画面に追加して使うと、より快適に利用できます。

---

## 技術構成

- Vite
- React
- TypeScript
- SVG（日本地図）
- localStorage（記録の保存）

---

## 注意事項

- シークレットモードでは記録が保持されない場合があります
- 端末変更時のデータ引き継ぎには対応していません
